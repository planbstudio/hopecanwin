# Hope Can Win

**A free counter-media briefing toolkit for Green Party supporters.**

Built by supporters, for supporters. Hope Can Win scans hostile right-wing media for attacks on the Green Party and its leadership, generates rebuttals, talking points, and a full social content kit — then packages it into a daily briefing anyone can use.

---

## What's in this toolkit

### [The Prompt](green-party-counter-media-prompt.md) — for anyone with Claude

A single copyable prompt you paste into [claude.ai](https://claude.ai) (free or Pro). No automation, no setup — just paste and run. You get:

- 7-outlet hostile media scan (GB News, Sun, Daily Mail, Telegraph, Express, Spectator, LBC)
- Story-by-story rebuttals with facts and quotable response lines
- 5–8 talking points for media appearances
- A full Supporters Content Kit (TikTok, Twitter/X, Instagram, Facebook)
- A priority action for the day

### [The Cowork Guide](green-party-cowork-setup-guide.md) — for automated daily briefings

A setup guide for Claude Cowork users who want the briefing to run automatically every morning and land in their inbox. Includes a complete SKILL.md template you install, configure with your own email recipients, and schedule. Same full briefing, delivered daily.

---

## Who this is for

Any Green Party supporter, local group, campaign team, or candidate who wants to:

- Know what the hostile media is saying about the Greens today
- Have ready-made rebuttals backed by facts
- Share professional social content across all platforms
- Stop being caught off-guard by media attacks

The hostile media monitoring works best from **MP, mayoral, and regional level upward** — national outlets don't typically run hit pieces on ward-level council candidates.

## How to use it

**Option 1 — The Prompt (zero setup)**
1. Open [the prompt file](green-party-counter-media-prompt.md)
2. Copy the prompt block
3. Paste it into a new conversation at [claude.ai](https://claude.ai)
4. Press Enter

**Option 2 — Cowork automation (15-minute setup)**
1. Open [the setup guide](green-party-cowork-setup-guide.md)
2. Follow Parts 1–3 to install the skill and configure your settings
3. Set up a daily schedule
4. Review and approve each briefing before it sends

## The methodology (v9)

This toolkit is based on a proven methodology that has been running successfully for several weeks:

- **7 search queries** across the main hostile outlets
- **5-day (120-hour) strict date window** — nothing stale
- **Anti-repetition** — every briefing contains only fresh material
- **[GP VOICE] fallback** — if fewer than 5 hostile stories, switches to proactive messaging
- **Full briefing always** — never a digest, never truncated
- **Supporters Content Kit** across all 4 social platforms, every time

---

## What this isn't

- This is **not an official Green Party product** — it's a supporter-made toolkit
- This is **not a bot or automated posting tool** — it generates content for humans to review, adapt, and share
- This **does not auto-send anything** — you always review and confirm before any email goes out
